Voici quelques commandes de base pour interagir avec une base de données MySQL :

! N'oubliez de lancer le programme mysql dans  XAMPP

**Version de Mysql** :

    mysql --version

**Connexion à mysql** :

    mysql -u root -p

**Deconnexion de mysql** :

    quit;
    ou
    exit;

**Affichage des bases de données** :

    SHOW DATABASES;

**Création d'une base de donnée** :

    CREATE DATABASE nom_base_de_donnee;


**Suppresion d'une base de donnée** :

    DROP DATABASE nom_base_de_donnee;


**Selectionde la base de donnée** :

    USE nom_base_de_donnee;
